#  AI-Powered College Chatbot (RAG + Groq LLM)

An intelligent chatbot designed to answer college-related queries using a **Retrieval-Augmented Generation (RAG)** approach. The system retrieves relevant information from a structured knowledge base and generates accurate responses using a Large Language Model.

---

##  Features

-  Interactive chatbot for college queries (courses, fees, admissions)
-  Retrieval-Augmented Generation (RAG) for accurate responses
-  Fast inference using Groq LLM (LLaMA 3)
-  Context-based answering from structured knowledge base
-  User-friendly interface built with Streamlit

---

##  Tech Stack

- **Language:** Python  
- **Frontend:** Streamlit  
- **LLM API:** Groq (LLaMA 3)  
- **Concepts:** RAG, NLP, Prompt Engineering  
- **Tools:** Git, GitHub  

---

##  How It Works

1. User enters a query in the chatbot  
2. The system retrieves the most relevant information from the knowledge base  
3. Retrieved context is passed to the LLM  
4. The model generates a context-aware response
